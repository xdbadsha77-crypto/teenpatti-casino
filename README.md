## گیم کے اصول 📚

Teen Patti میں ہر کھلاڑی کو 3 کارڈ ملتے ہیں:

- **Trail/Trips** - تینوں کارڈ ایک جیسے (سب سے اچھا)
- **Pure Sequence** - تین متوالی کارڈ (ایک جیسے رنگ)
- **Sequence** - تین متوالی کارڈ (مختلف رنگ)
- **Flush** - تینوں کارڈ ایک جیسے رنگ
- **Pair** - دو کارڈ ایک جیسے
- **High Card** - کوئی ملاپ نہیں (سب سے برا)
